# 🌍 COVID-19 Global Data Visualization Dashboard (Tableau)

An interactive Tableau dashboard designed to analyze global COVID-19 cases, deaths, and population infection trends across countries and continents using real-world data.

---

## Overview
This project presents a comprehensive data visualization dashboard built using Tableau to explore the global impact of COVID-19. The dashboard enables users to analyze total cases, total deaths, death percentage, and population infection metrics at both continental and country levels.

---

## Tech Stack
- Tableau Desktop
- Tableau Public
- Excel / CSV (data preprocessing)
- Data Visualization & Analytics

---

## Data Source
**Source:** Publicly available COVID-19 datasets (global-level data)

The dataset includes country-wise and continent-wise information such as:
- Total confirmed cases
- Total deaths
- Death percentage
- Population infected percentage
- Date-wise trends

The data was cleaned and structured to support accurate aggregation and visualization.

---

## Key Features & Insights

### Global KPIs
- Total COVID-19 cases worldwide
- Total deaths across all regions
- Global death percentage

---

### World Map Visualization
- Interactive world map highlighting countries based on infection intensity
- Color gradient represents percentage of population infected
- Enables quick identification of high-impact regions

---

### Continent-wise Analysis
- Bar chart comparing total deaths across continents
- Highlights regional severity and comparative impact

---

### Time-Series Analysis
- Line chart visualizing average population infected over time
- Enables trend analysis and comparison across countries
- Helps understand the progression of infection rates

---

## Business & Analytical Impact
- **Public Health Insights:** Enables clear visualization of regional and global COVID-19 impact  
- **Trend Analysis:** Helps identify infection surges and comparative growth across regions  
- **Data Storytelling:** Transforms raw pandemic data into actionable visual insights  
- **Decision Support:** Useful for researchers, analysts, and policymakers
